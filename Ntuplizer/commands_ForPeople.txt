This is to  run data with JEC that are run dependent:

python submit_all_opt.py -d CRAB_JEC_V6 -c config_generic_opt.py -f samples/Fall2017/Dataset_SingleMuon_2017.txt -D True -l True -s ""



For MC, no need to use run dependent corrections:

python submit_all.py -d CRAB_JEC_V6 -c config_generic.py -f samples/Fall2017/MC2017_v2.txt  -s ""

python submit_all.py -d CRAB_JEC_V6 -c config_generic.py -f samples/Fall2017/MC2017_v2_ext.txt  -s "_ext"



To check the status and resubmit:

ls CRAB_JEC_V6/* |grep "_JEC_V6"  | awk {' print "crab resubmit " $0 }' |sed 's/:/ /' >resubmit.sh 
ls CRAB_JEC_V6/* |grep "_JEC_V6"  | awk {' print "crab status " $0 }' |sed 's/:/ /' >status.sh

chmod +x  status.sh
chmod +x  resubmit.sh

./status.sh
./resubmit.sh
